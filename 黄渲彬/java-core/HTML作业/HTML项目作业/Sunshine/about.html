公司简介

阳光家园房地产(集团)股份有限公司成立于2000年，是中国
阳光家园房地产集团控股的大型房地产企业，国家一级房地
产开发资质企业，国有房地产企业综合实力榜首，并连续四
年蝉联央企房地产品牌价值前列，2013年，公司品牌价值达
90.23亿元。2016年，公司股票在上海证券交易所上市，2019
年公司实现销售签约433.82亿元。截至2020年一季度，公司
总资产将超千亿。
